Table Tennis Cup


In this project what I built is a website to manage a Table Tennis Cup, it helps to keep track of everything including match dates, scores, players contact info, and players handicaps
It can be used to calculate the final scores of matches as the player handicaps have to be taken into account and the calculations are not trivial.

The tools I used to build it are Vue.js for the front end, Express.js (Node.js) for the backend and finally MySQL for the database. If you would like to view the site you can find it here:
